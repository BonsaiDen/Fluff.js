Fluff.js
========

**Fluff.js** is a 2D Game Engine created using SFML, OpenGL and V8.


Compiling
---------

| First make sure you got V8 and SFML(and I guess OpenGL).  
| Then just run the make script, you can also pass `-r` for optimizations.
